I’ve never been very good about blogging. And now that I’ve got a full-time job, it will probably be even harder. I’m going to try to commit to at least one blog post, a week, though. I think if I keep with a theme, things will be a bit easier. I’m calling this theme “Programming and Productivity” until I come up with a better name. There are two basic ideas, though (as illustrated in the title).

I want to keep this blog simple, so I’m using Markdown for all of my posts. I picked posterous because it appears to be the only free platform that has all of the features that I want. If you have a better solution, though, please let me know.

Programming

Any interesting programming projects I’m learning. New languages I’m looking into. Blog posts about programming that I just want to drop a link to.

I’ve already got some new ideas that I want to post about. For instance:

  • I want to learn LISP
  • I’m trying to come up with useful applications for things like Meteor and Go.
  • I’ve got a few web applications that I want to work on

Productivity

Anything that I find helps me when I’m at work or home. The idea is things that make my life easier. This could be a wide variety of things:

  • An excellent app that I’ve found.
  • A good website for keeping track of things.
  • An interesting vim tip that I learn.

There may be other things that I want to talk about. Random posts about Christmas, work, or home. But the general trend of the blog should hopefully stay on the course of Programming and Productivity.

For next week, I think I’m going to try to have a post about learning Lisp ready to go. If it’s not ready, I swear I’ll have something.
